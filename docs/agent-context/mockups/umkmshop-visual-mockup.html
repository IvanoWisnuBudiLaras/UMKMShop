Implementation annotation

Mockup ini sengaja memakai flat shape, border lembut, tanpa glassmorphism, tanpa parallax, tanpa shadow bertumpuk. Semua frame motion adalah storyboard statis agar guideline mudah diterjemahkan ke Compose.

Spacing utama 16dp, komponen radius 8dp, CTA min-height 48dp, dan ikon aktif memakai fill sebagai state.

Visual artifact only. This file does not alter Android app source, data model, route flow, Supabase behavior, or business logic.
