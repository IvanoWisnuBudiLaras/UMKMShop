Catatan: dark mode di kolom Mode Penjual pada mockup ini murni untuk mendemonstrasikan token warna dark dari Design.md §3.1 dalam satu gambar. Di aplikasi sebenarnya, dark/light mode adalah preferensi sistem/pengguna yang berdiri sendiri — tidak berpindah otomatis saat toggle Pembeli ⇄ Penjual. Kedua mode (Pembeli, Penjual) harus bisa tampil di light maupun dark, tergantung pengaturan device masing-masing pengguna.
